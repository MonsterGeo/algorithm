## 栈

栈上的插入操作称为压栈(push)，而删除元素的操作称为弹出(pop)。给定一个数组```S[1..n]```，他表示一个容纳n个元素的集合。该数组存在一个属性```S.top```，访问它将返回S最新插入的元素。换句话说，该数组为```S[1..S.top]```，当```S.top = 0```，表示栈不含任意元素。即栈为空。若此时继续执行```pop```，会发生栈下溢。报错，若```S.top```超过```n```，则发生上溢。

我们通过下述伪代码实现栈（不考虑任何异常问题处理）

```
STACK-EMPTY(S)
1| if S.top == 0
2|  return TRUE
3|  else return FALSE
```

```
PUSH(S,x)
1| S.top = S.top+1
2| S[S.top] = x
```

```
POP(S)
1|  if STACK-EMPTY(S)
2|      error "underflow"
3|  else S.top = S.top-1
4|      return S[S.top+1]
```

## 队列

和栈不一样，队列遵循先入先出的性质。一个队列具有队头和队尾。当一个元素入队时，它被放在队尾的位置。那么，数组Q[1..n],它具有属性```Q.head```和```Q.tail```，即队列队尾。当```Q.head = Q.tail```时，队列为空，初始化时，```Q.head = Q.tail = 1```。当```Q.head = Q.tail+1```表示队列是满的。

```
ENQUEUE(Q,x)
1| Q[Q.tail] = x
2| if Q.tail == Q.length
3|    Q.tail = 1;
4| else Q.tail = Q.tail+1
```

```
DEQUEUE(Q)
1| x = Q[Q.head]
2| if Q.head = Q.length
3|  Q.head = 1
4| else Q.head = Q.head+1
5| return x
```
